# ✨ My Cosmos
```
당신의 계절에서, 별자리를 찾아보세요.
```

밤하늘 보는 것을 좋아하던 제가 고등학교 때 기획했던 게임인데, 2022 졸업 프로젝트로 개발하게 되었습니다.  

이 게임은 사계절의 챕터로 이루어져있으며, 각 계절을 대표하는 별자리들을 찾는 퍼즐 게임입니다.

밤하늘에 떠있는 별을 터치해 선으로 잇고, 별자리를 만들 수 있습니다.

이 게임을 통해 보다 쉽게 별자리를 공부해보세요.
