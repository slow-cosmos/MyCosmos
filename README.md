# My Cosmos
: 별자리 찾기 퍼즐 게임

## 개발 기간
2022년 3월 17일 ~

## 플레이 방법
### 조작
+ WASD, 방향키 → 카메라 회전  
+ 마우스 휠 → 줌 인, 줌 아웃  
+ 마우스 왼쪽 클릭 → 별 선택/취소, 선 제거  
### 게임 방법
+ 오른쪽 아이콘을 눌러 찾아야 할 별자리 확인
+ 각 별자리 이름을 누르면 별자리의 모양과 설명이 나옴  
+ 두 번째 탭을 누르면 전체 별자리의 위치를 볼 수 있음  
+ 한 챕터의 모든 별자리를 찾으면 챕터가 클리어 됨
